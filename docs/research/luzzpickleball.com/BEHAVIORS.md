# LuzzPickleball Homepage Behaviors

## Global

- The top announcement bar cycles through short promotional messages.
- The header is transparent over the hero and stays visually light while the hero is in view.
- The page uses bold display typography for section titles and a cleaner body font for product copy and paragraph text.

## Hero

- The hero is a time-driven carousel with three slides.
- Slides switch automatically on an interval and can also be advanced manually with previous/next controls.
- The active slide includes the CTA inside the artwork itself, so the imagery is composited rather than built from separate HTML text layers.

## Testimonial Marquee

- Reviewer quotes move horizontally in a loop.
- Each item has an avatar, a quote, and a source label.

## Spotlight Paddles

- The interaction model is click-driven tabs.
- Tabs switch between `Pro`, `Performance`, and `Infinity`.
- Each tab exposes a horizontal/scrollable product list on the source site; the clone keeps the tab interaction and visible card content.

## Luzz Go

- Static editorial section.
- No click-driven state on the source homepage.

## Warranty

- Static promotional banner with a single CTA.
- Visual interest comes from the repeating circular background pattern rather than animation.

## Video Reviews

- The source site uses a carousel-like track.
- Cards emphasize thumbnail, play affordance, quote, and reviewer identity.

## Collection Grid

- Hover states slightly lift cards and scale imagery.
- Layout changes from a five-tile mosaic on desktop to a vertical stack on mobile.

## Universal Collaboration

- Static split layout with two linked promotional banners.
- Hover scales the banners slightly.

## FAQ

- Click-driven accordion.
- Only the summary rows are visible by default.

## Team

- Horizontal drag/scroll gallery on desktop.
- Cards stack into a scrollable row on small viewports.
